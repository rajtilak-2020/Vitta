# Vitta

A simple, elegant personal finance tracker android app designed for speed and clarity. Track every Rupee with ease.

## Features

- **Quick Entry**: Add expenses and income in seconds.
- **Running Balance**: See your updated balance instantly.
- **Clean UI**: A minimalist, modern interface that’s easy on the eyes.
- **Secure**: All data is stored locally and securely on your device.

## Getting Started

### Prerequisites

- Flutter SDK 3.12.2 or higher.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/rajtilak-2020/Vitta.git
   ```

2. Navigate to the project directory:
   ```bash
   cd vitta
   ```

3. Install dependencies:
   ```bash
   flutter pub get
   ```

## Usage

Run the application on your connected device or emulator:

```bash
flutter run
```
